ToDogether

It is a full-stack mobile productivity application designed to help parents encourage positive habits and task completion in children through gamification.
Parents can assign tasks and habits, monitor progress, and track completion statistics, while children earn points by completing activities. These points can then be used within an interactive virtual aquarium where users can purchase, feed, and maintain virtual fish.

A detailed project report containing the analysis, design process, database architecture, implementation details, testing, and screenshots is included in this repository.

## Tech Stack
- React Native (Expo)
- Django REST Framework
- Python
- SQLite
- Celery & Redis
